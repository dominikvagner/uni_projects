PŘEKLAD:
Pro překlad byl zvolen nástroj Maven.
Příkazy pro překlad a spoušťení se musí zadávat ze složky projektu. (Tam kde je pom.xml soubor)

- Příkaz pro překlad a vygenerování jar archivu:
    mvn package

- Příkaz pro vygenerování dokumentace:
    mvn javadoc:javadoc

SPOUŠTĚNÍ:
Překladem vytvorříme také jar archiv, a ten následně spouštíme.

- Příkaz pro spuštění:
    java -jar dest/ija-app.jar
